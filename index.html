    Que p*** é o Promto ?

    Promto e um software de identificação, classificação e controle de itens de estoque, a principal função e criar codificação de itens subdivido em Empresas, Tipos, Famílias, Sub-familias e modelos, e ainda terminar com uma codificação humanamente legível de fácil identificação e memorização, a partir da codificação inteligente o algoritmo do programa gera uma pré-estrutura dos produtos acabados com base nas suas classificações.

    Como faço esse c**** funcionar ?

    Primeiro é preciso instalar todas as dependências:

    • AHK
    • MariaDB
    • Java
    • GitHub for windows
    • Git

    depois de baixar e instalar todas as dependências, crie um novo banco de dados e configure o arquivo settings.json para apontar para o devido banco de dados, todas as demais tabelas serão criadas quando o programa for iniciado pela primeira vez, o github para windows e o git servem para atualizar o programa para a versao mais nova, não são obrigatórios.
